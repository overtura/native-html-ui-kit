Native HTML UI Kit
메뉴 폼 입력 탐색 대화상자

브라우저가 이미 가진 UI 계약을 다시 엽니다

popover, details name, command/commandfor, dialog, form validation을 JavaScript 없이 조합하는 한국어 UI kit 실험입니다.

Menus & Popovers

메뉴 버튼, 팝오버, 링크 목록을 HTML 속성으로 연결합니다.

프로필 메뉴

계정 설정 확인 dialog 맨 위로

Forms

브라우저 기본 validation과 CSS 상태 선택자로 입력 흐름을 만듭니다.

검증 폼

required와 minlength는 브라우저 기본 검증으로 처리됩니다.

Datalist Search

검색 입력과 datalist 후보를 묶어, 작은 컴포넌트 탐색기를 JavaScript 없이 제공합니다.

컴포넌트 찾기

입력을 시작하면 브라우저가 datalist 후보를 표시합니다. 값이 비어 있으면 native validation이 동작합니다.

Dialog & Accordion

지원 브라우저에서는 commandfor로 dialog를 선언적으로 열 수 있습니다.

접근성

네이티브 요소가 기본 keyboard interaction과 이름을 제공합니다.

유지보수

중앙 maintainer bot은 컴포넌트 예제와 문서를 작은 PR로 개선합니다.
